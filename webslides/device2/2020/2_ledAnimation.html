コメント

ソースコードに注釈をつける。
  • //を先頭につけた行は、コンパイル時に無視される。
  • 複数行コメントは、/*と*/で囲む。
//数値表示用変数の宣言
let num = 0
//数字を1から順番に表示し続ける
basic.forever(function() {
  /* "num++"は、、引数に代入した後に
  インクリメントすることを表す */
  basic.showNumber(num++);
})

電子回路の試作

ブレッドボードを用いて、センサやモータなどの回路を組み立てる。
  • ブレッドボードははんだ付けが不要。
  • 回路を構成する素子を差し込む穴がある。
  • 右図の線で表された部分が内部で繋がっている。

例題８

LEDを点灯する回路を作ろう。

  • Arduino "5V"ピン > ブレッドボード > 抵抗（330Ω）> LED > Arduino "GND"ピン
  • LEDは流せる電流の向きが決まっており、+側をアノード、-側をカソードと呼ぶ。
  • LEDの足の長い方がアノード。

電子回路でよく登場する用語

  • 電圧

    電気を流す力の大きさ。ボルト。

  • 電流

    回路に流れている電気の量。アンペア

  • 抵抗

    電流の流れにくさ。オーム。または電流を調整する素子。

  • GND（グラウンド）

    電源の-（マイナス）極のこと。

補足

Arduinoでよく用いられる関数

  • void pinMode(ピン番号, 入出力);

    引数のピンを入力（INPUT）か出力（OUTPUT）に設定する。

  • int digitalRead(ピン番号);

    ピンに入る電圧をHIGHで返す。

  • void analogWrite(ピン番号, 状態);

    引数で指定したピンから0-255のPWM出力をする。

  • void digitalWrite(ピン番号, 状態);

    引数のピンをHIGH (5v)かLOW (0v)にする。

  • void analogRead(ピン番号);

    ピンに入る電圧を0-255（0-5v）の整数値で返す。

  • void delay(ミリ秒);

    引数の値だけ何もしないで待つ。
