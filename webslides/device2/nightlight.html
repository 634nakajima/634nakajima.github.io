関数 とは

  • コンピュータに対する命令、処理。
  • コンピュータに命令することを関数の実行と呼ぶ
  • 実行できる関数は、あらかじめ定義（用意）された関数と、自分で定義する関数に分けられる。
関数の実行
int  variable = function(arg);

関数の定義
int function(int argument) {
    subfunction(arg);
    return x;
}
